Sistema de Gestão de Coworking

Uma aplicação para gerenciar o Coworking da sua empresa.

A palavra Coworking - traduzindo do inglês para o português - significa Espaço de Trabalho Compartilhado, ou seja, um ambiente para se trabalhar com outras pessoas, por este motivo, esse sistema foi desenvolvido para o gerenciamento desse Coworking.
